## What is this?

Forsyth Games Portal is a modern educational gaming website for Forsyth County Schools. It provides a curated collection of browser games and utilities, designed for speed, usability, and a professional school experience. The portal features a premium dark mode interface, fast search, and robust access controls to ensure safe and appropriate use during school hours and within Georgia, USA.

Built with Next.js and React, Forsyth Games Portal is fully responsive and optimized for all devices. No personal data is collected; all games run locally in your browser.
