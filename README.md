## ğŸ› ï¸ Development Setup

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager

### Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Setup static files (IMPORTANT!):**
   ```bash
   ./setup.sh
   ```
   This copies game files to the `public/` directory where Next.js can serve them.

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

5. **Build for production:**
   ```bash
   npm run build
   npm start
   ```

## ğŸ® Adding New Games

1. Add game files to `/games/your-game/`
2. Include a `favicon.png` in the game directory
3. Add game data to `data/games.ts`:
   ```typescript
   {
     id: 'your-game',
     name: 'Your Game',
     category: 'ACTION',
     iconUrl: '/games/your-game/favicon.png',
     iframeSrc: '/games/your-game/'
   }
   ```
4. Run `./setup.sh` to copy to public directory
